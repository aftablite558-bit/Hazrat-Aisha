# Hazrat Aisha Academy - School Management System

A comprehensive, modern school management system built for Hazrat Aisha Academy. This application provides a unified platform for school administration, academic management, and public information.

## Features

- **Public Portal:** Home, About, Academics, Gallery, Notices, Contact, and Result verification.
- **Student Management:** Enrollment, records, and detailed profiles.
- **Staff Management:** Teacher and administrative staff records.
- **Attendance Tracking:** Daily attendance logging for students and staff.
- **Fee Management:** Fee collection, receipts, and outstanding dues tracking.
- **Examination System:** Exam scheduling, marks entry, and report card generation.
- **System Administration:** General settings, user roles (RBAC), security policies, backup, and system logs.

## Tech Stack

- **Frontend:** React 19, TypeScript, Vite, Tailwind CSS, shadcn/ui, motion
- **Backend:** Firebase (Authentication, Realtime Database, Storage)
- **Deployment:** Vercel / Netlify / Firebase Hosting / Google Cloud Run

## Getting Started

1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Configure your `.env` file based on `.env.example`.
4. Run `npm run dev` to start the development server.
